# HealthAgents

HealthAgents is a Multi-Agent Medical Intelligence Platform built for the AMD Developer Hackathon.

## Features

- Multi-Agent AI
- Medical RAG
- Medical Report Analysis
- Medical Image Understanding
- Symptom Analysis
- FastAPI Backend
- Streamlit Frontend

## Tech Stack

- Python
- FastAPI
- Streamlit
- LangGraph
- LangChain
- Fireworks AI
- FAISS
